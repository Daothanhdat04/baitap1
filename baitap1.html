HTML Overview

HTML Basic

HTML (HyperText Markup Language) is used to create webpages. It consists of various elements and attributes used to structure and format content.

HTML Elements

An HTML element typically consists of a start tag, content, and an end tag. Example:

<tag>content</tag>

HTML Attributes

Attributes provide additional information about HTML elements. They are added to the start tag. Example:

<tag attribute="value">

HTML Headings

HTML provides six levels of headings. Example:

<h1>This is a heading</h1>
<h2>This is a sub-heading</h2>
<h3>This is a smaller heading</h3>

HTML Paragraphs

The <p> tag is used to define a paragraph. Example:

<p>This is a paragraph.</p>

HTML Styles

Inline CSS styles can be added using the style attribute. Example:

<tag style="property:value">content</tag>

HTML Formatting

HTML provides tags for text formatting. Examples:

<strong>Bold text</strong>
<em>Italic text</em>
<u>Underlined text</u>

HTML Quotations

For longer quotations, use the <blockquote> tag. For shorter inline quotations, use the <q> tag. Example:

<blockquote>This is a blockquote.</blockquote>
<q>This is an inline quote.</q>

HTML Comments

Comments are not displayed in the browser. Syntax:

<!-- This is a comment -->

HTML Colors

Colors can be defined by names, HEX codes, RGB, or HSL values. Example:

<div style="color: #ff0000;">This is red text.</div>

HTML Links

Links are created with the <a> tag. Example:

<a href="https://www.example.com">This is a link</a>

HTML Images

Images are added using the <img> tag. Example:

<img src="image.jpg" alt="Description">

HTML Tables

Tables are created with the <table> tag. Example:

<table>
  <tr><th>Header 1</th><th>Header 2</th></tr>
  <tr><td>Row 1, Cell 1</td><td>Row 1, Cell 2</td></tr>
  <tr><td>Row 2, Cell 1</td><td>Row 2, Cell 2</td></tr>
</table>

HTML Lists

HTML supports ordered (<ol>) and unordered (<ul>) lists. Example:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

HTML Block & Inline

Block-level elements start on a new line and take up the full width, while inline elements do not start on a new line. Examples:

<div>This is a block-level element.</div>
<span>This is an inline element.</span>
